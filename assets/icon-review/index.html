GoatCitadel Mobile

Icon Direction Review

The mobile repo is already wired to custom icon assets, but the current mark still feels busy and uneven at launcher scale. These alternatives take your new brief literally: a castle or citadel structure with a clear G integrated into the icon.

Current state The app is configured correctly in `app.json`; the remaining question is design quality, not missing wiring.
Design problem The current icon mixes shield, glow, horn texture, and interior detail in a way that muddies the small-size read.
Goal Pick a direction that feels literal, legible, and more app-icon-native before we replace the live PNG assets and rebuild the app.
